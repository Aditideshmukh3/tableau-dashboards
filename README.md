# Tableau Dashboards Portfolio

A collection of interactive Tableau dashboards created for data analytics and business intelligence practice.  
This repository will be updated with additional dashboards over time.

## Dashboards Overview
- **Superstore Sales Performance** – Sales and profit analysis using Tableau Sample Superstore data  
- **Netflix Content Analysis** – Analysis of movies and TV shows by release trends and ratings  
- **Customer Churn Analysis** – Understanding customer churn patterns and influencing factors  
- **Regional Sales Analysis** – Region-wise and segment-wise sales performance insights  
- **Boston Housing Price Analysis** – Factors affecting housing prices using socio-economic data  

## Tools & Technologies
- Tableau  
- Git & GitHub  
- VS Code  

## Repository Structure
Each dashboard folder contains:
- Dataset (CSV/XLSX)
- Tableau workbook (.twb/.twbx)
- Dashboard export (PDF)
- Documentation
